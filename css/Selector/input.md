input
=============
사용자가 입력할 부분은 주로 `<input>` 태그를 이용해 넣습니다.

## input
`<input>` 태그에서 입력 형태를 지정할 때 `type` 속성을 사용합니다. 다양한 `type` 속성값을 검색하면 나옵니다.

로그인 창에서 아이디와 비밀번호를 입력하거나 로그인 버튼을 클릭하는 등 `<input>` 태그는 매우 다양한 입력 형태로 만들 수 있습니다.

```html
<input type="text"> <!-- 텍스트 -->
<input type="password"> <!-- 비밀번호 -->
```
